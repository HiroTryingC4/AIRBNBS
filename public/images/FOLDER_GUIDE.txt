📏 Image Sizes:
   • Hero: 1920x1080px (landscape)
   • Gallery: 1200x800px (landscape)
   • Thumbnails: 800x600px

💾 File Size:
   • Target: 200-500KB per image
   • Compress at: tinypng.com

📝 File Naming:
   • Use lowercase with hyphens
   • Be descriptive
   • Example: living-room-sunset.jpg

🔄 How to Use:
   1. Add photos to folders above
   2. Update lib/mockData.ts
   3. Update app/gallery/page.tsx
   4. Update app/page.tsx (hero)
   5. Run: npm run build
   6. Deploy!
